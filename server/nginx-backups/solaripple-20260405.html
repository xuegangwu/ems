群体智能数字孪生

EnOS 平台将物理电站映射为实时数字孪生体。LSTM 神经网络预测 48 小时光伏出力，结合储能 SOC 动态调度策略，实现峰谷套利收益最大化。

  • 3D 可视化 + 实时 SVG 能量流
  • LSTM 预测：光伏/负荷 48h 超前规划
  • 储能 AI 调度：SOC 最优充放策略
  • InfluxDB 时序数据，5 分钟分辨率

虚拟电厂（VPP）IEC 104 交易

Master 平台模拟 IEC 60870-5-104 协议，实时接收电网 AGC 调度指令。储能作为灵活资源参与调频辅助服务市场，秒级响应频率偏差。

  • IEC 104 M_ME_TF_1 / C_SE_TF_1 协议
  • AGC 自动调度信号，5 秒刷新
  • 日/月/年收益全链路报告
  • 支持多储能资源聚合
🎛️ VPP

Pretext — 极速渲染引擎

自研 Pretext 文本渲染引擎，通过一次性 Canvas 预测量 + 纯算术分行，替代传统 DOM offsetWidth 测量。SVG 标签宽度预计算，零 DOM 访问，零回流抖动。

  • Gzip 压缩后仅 1.2KB，零依赖
  • prepare() + layout() 两阶段 API
  • SVG / ECharts / Recharts 通用
  • GitHub 开源：xuegangwu/pretext
